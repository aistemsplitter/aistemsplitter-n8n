# AIStemSplitter n8n Community Node

Community automation integrations for AIStemSplitter, published as the n8n
community node package `@aistemsplitter/n8n-nodes-aistemsplitter`.

- Homepage: https://aistemsplitter.org
- API docs: https://aistemsplitter.org/developers/api
- API base: `https://api.aistemsplitter.org/v1`

The n8n node includes:

- Get Credits
- Create Split
- Get Split

Install it in a self-hosted n8n instance from the Community Nodes screen with:

```text
@aistemsplitter/n8n-nodes-aistemsplitter
```

Additional source-ready artifacts:

- Pipedream app definition and Get Credits, Create Split, and Get Split actions.
- Zapier manual submission notes.
- Make manual submission notes.

n8n verified-node listing, Pipedream, Zapier, and Make publication still require
their respective platform accounts and review flows.
