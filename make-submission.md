# Make submission notes

Make publication is manual and blocked until Make app/review access is
available.

Prepared integration scope:

- Connection: API key header, `Authorization: Bearer <api_key>`.
- Create Split module: `POST https://api.aistemsplitter.org/v1/audio/splits`.
- Get Split module: `GET https://api.aistemsplitter.org/v1/audio/splits/{splitId}`.
- Public docs: https://aistemsplitter.org/developers/api.

Submission blockers:

1. Create or access the Make app.
2. Add the API key connection.
3. Configure Create Split and Get Split modules.
4. Test with a real API key.
5. Submit for Make review.
